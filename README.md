# Admin Dashboard

A full-stack admin dashboard application with NestJS backend API and Angular frontend UI.

## Project Structure

This repository contains two main applications:

### Backend
- **Location**: `admin-dashboard-api/`
- **Framework**: NestJS
- **Language**: TypeScript
- **Port**: 3000 (default)

### Frontend
- **Location**: `admin-dashborad-ui/`
- **Framework**: Angular
- **Language**: TypeScript

## Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Backend Setup
```bash
cd admin-dashboard-api
npm install
npm run start
```

### Frontend Setup
```bash
cd admin-dashborad-ui
npm install
npm start
```
